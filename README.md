# 本地视频仓库

一个功能强大的本地视频管理系统，支持视频扫描、标签管理、关键帧提取、收藏、历史播放等功能。

## 功能特性

### 🎬 视频管理
- 自动扫描本地视频文件
- 支持多种视频格式
- 自动生成视频缩略图
- 视频信息详情展示

### 🏷️ 标签系统
- 自动为视频添加文件夹标签
- 自定义标签管理
- 按标签筛选视频
- 标签颜色区分

### ⭐ 关键帧功能
- 视频关键帧提取
- 关键帧时间轴展示
- 关键帧快速定位
- 关键帧备注功能

### ❤️ 收藏系统
- 视频收藏功能
- 收藏视频快速访问

### 📺 历史播放
- 最近播放记录
- 播放次数统计

### 🗑️ 回收站
- 视频软删除
- 已删除视频管理
- 视频恢复功能

### 🎨 主题支持
- 多种主题配色
- 主题持久化

### 📱 响应式设计
- 支持桌面端和移动端

## 技术栈

- **前端**: Vue 3 + Element Plus + Vite
- **后端**: Express.js + Node.js
- **数据库**: SQLite (sql.js)
- **视频处理**: FFmpeg

## 安装与部署

### 环境要求
- Node.js 16+
- FFmpeg (用于视频处理和缩略图生成)

### 安装步骤

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd video-repo
   ```

2. **安装依赖**
   ```bash
   # 安装后端依赖
   npm install
   
   # 安装前端依赖
   cd client
   npm install
   cd ..
   ```

3. **配置FFmpeg**
   - 确保FFmpeg已安装并添加到系统PATH
   - 或在`server/scanner.js`中配置FFmpeg路径

4. **构建前端**
   ```bash
   cd client
   npm run build
   cd ..
   ```

5. **启动服务器**
   ```bash
   npm run server
   ```

6. **访问应用**
   打开浏览器访问 `http://localhost:3001`

## 使用指南

### 首次使用
1. 点击"设置"按钮
2. 添加视频扫描路径
3. 点击"初始化项目"
4. 点击"扫描新增视频"

### 基本操作
- **播放视频**: 点击视频卡片或文件名
- **添加标签**: 在播放页面的标签管理部分
- **添加关键帧**: 在播放页面点击"添加关键帧"按钮
- **收藏视频**: 点击视频卡片上的"收藏"按钮
- **删除视频**: 点击视频卡片上的"删除"按钮
- **查看历史**: 点击首页的"历史播放"按钮
- **查看已删除**: 点击首页的"已删除"按钮

### 高级功能
- **主题切换**: 在设置页面选择主题
- **批量扫描**: 点击"扫描新增视频"按钮
- **路径管理**: 在设置页面管理扫描路径

## 常见问题

### 视频扫描失败
- 检查FFmpeg是否正确安装
- 检查扫描路径是否存在且有访问权限
- 检查视频文件格式是否支持

### 缩略图生成失败
- 确保FFmpeg可以正常工作
- 检查视频文件是否损坏

### 历史播放不显示
- 确保视频播放时已加载完成
- 检查浏览器控制台是否有错误
